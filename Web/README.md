# Basic file structure is as follows:

* Web/
  * run.py
  * WhoKnows/
    * \_\_init__.py
    * models.py
    * views.py
    * static/
      * (CSS files)
      * (JS files)
      * (Any other non-changing files)
    * templates/
      * index.html
      * layout.html
      * profile.html

# Hardcoded files
Please place Hardcoded files in a new folder in the _web_ directory with the page's name.
